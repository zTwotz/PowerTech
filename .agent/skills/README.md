# Antigravity Open Skills Bundle

This bundle follows the lightweight open skill format:
each skill is a folder containing a `SKILL.md` file with YAML frontmatter and Markdown instructions.

Included skills:
- project-context
- ui-style-system
- component-library
- aspnet-mvc-architecture
- database-domain-rules
- ecommerce-business-rules
- implementation-workflow
- navigation-screen-priority
- testing-qa-review
- agent-execution-rules

Suggested loading:

- Backend tasks: project-context + aspnet-mvc-architecture + database-domain-rules + ecommerce-business-rules + implementation-workflow
- Review tasks: testing-qa-review + agent-execution-rules + relevant feature skills
